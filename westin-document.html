Media Screen

Bootstrap includes a responsive, mobile first fluid grid system that appropriately scales up to 12 columns as the device or viewport size increases. But sometimes text, images or ... don't work good in some screens. Because of this web designers and developers use "Media Screens" in CSS files to solve this problem. In westin template CSS we use some media screens to make our theme fully responsive for all screens. You can see an example from "style.css" of westin template below. 


   @media (max-width: 991px){
     //Media Of Components(max-width: 991)//
     .title-box h3 {
        font-size: 52px;
    }
    .row-sticky {
        display: block;
    }
    .col-sticky {
        position: inherit;
    }
    .return-to-top{
        right: 0;
        border-radius: $rounded-sm 0 0 $rounded-sm;
    }
    //Media Of Hero(max-width: 991)//
    .hero-03 {
        padding-bottom: 8rem;
        .personal-img img{
            width: 200px;
        }
    }
    .title-box h6{
        letter-spacing: 7px;
    }
    //Media Of Counted(max-width: 991)//
    .counted-box-02{
        margin-top: 24px;
    }
    }
